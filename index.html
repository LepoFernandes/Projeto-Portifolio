Sobre mim

Olá, eu sou Lucas. Tenho 24 anos, minha carreira começou no setor de varejo, onde ao longo dos anos, desenvolvi habilidades robustas em gestão, tomada de decisões e resolução de problemas enquanto atuava como empreendedor em um mercado e bar.

Recentemente, decidi embarcar em uma empolgante transição para a área de Engenharia de Software. Acredito que as habilidades adquiridas durante minha experiência como empreendedor podem ser aplicadas de maneira inovadora no contexto tecnológico.

Atualmente estou cursando Engenharia de Software pela Estácio e muito ansioso para explorar novos horizontes na área, aplicando minha paixão por aprender e meu histórico empreendedor ao mundo tecnológico.

HTML (Hypertext Markup Language) é a linguagem padrão para a criação e design de páginas web.

CSS

CSS (Cascading Style Sheets) é usado para estilizar o layout de documentos HTML.

JavaScript

JavaScript é uma linguagem de programação amplamente utilizada para desenvolvimento web interativo.
